Why Choose PECU?

The Proxmox-Enhanced-Configuration-Utility (PECU) is a comprehensive tool designed to simplify GPU passthrough and the overall management of Proxmox VE. Whether you are setting up GPU passthrough for the first time or managing multiple configurations, PECU provides an intuitive interface that guides you through each step, ensuring a smooth and optimized experience.

Get Started with PECU

How to Install and Use PECU

  1. Clone the Repository: Run git clone https://github.com/Danilop95/Proxmox-Enhanced-Configuration-Utility.git.
  2. Set Permissions: Use chmod +x proxmox-configurator.sh to make the script executable.
  3. Run the Script: Execute sudo ./proxmox-configurator.sh and follow the interactive prompts to configure your Proxmox VE environment.
  4. Enjoy a Fully Optimized Setup: Benefit from automated GPU passthrough configuration and efficient package management.
