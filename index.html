Overview

The Proxmox-Enhanced-Configuration-Utility (PECU) is a powerful Bash script designed to simplify the configuration and management of Proxmox VE environments. This utility helps automate the setup of GPU passthrough and manages package repositories efficiently, allowing you to optimize your Proxmox setup effortlessly.

Features

  • Automated GPU passthrough setup for better VM performance.
  • Easy management of Proxmox package repositories.
  • System configuration checks for optimal hardware settings.

Installation

To get started with PECU, follow these simple steps:

  1. Clone the repository from GitHub.
  2. Run the script directly or locally on your Proxmox server.

For detailed instructions, visit the GitHub Repository.
